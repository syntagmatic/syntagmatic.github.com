Future Tools

If you have an inclination to build the next generation of data science tools like Google Refine, but on the web, check out these projects:

  • Data-Driven Documents (d3) - A general-purpose visualization toolkit
  • Crossfilter - Fast multidimensional filtering for coordinated views
  • Parallel Coordinates - A visual toolkit for multidimensional detectives (disclaimer: my project)
  • Parallel Sets - A visualisation technique for multidimensional categorical data
  • SlickGrid - A lightning fast JavaScript grid/spreadhseet
